# 📁 Smart File Organizer

## 🚀 Overview

A web application built using Python (Flask), HTML, and CSS that allows users to upload files and automatically organizes them into folders based on file type.

---

## 🌐 Live Demo

https://files-organizer.onrender.com

---

## ✨ Features

* Upload files through browser
* Automatic file classification (Images, Documents, Videos, Others)
* Organized folder structure
* File statistics display
* Clickable files to view/download

---

## 🛠️ Tech Stack

* Python (Flask)
* HTML, CSS

---

## ▶️ Run Locally

```
pip install flask
python app.py
```
